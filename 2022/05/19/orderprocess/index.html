PCB/PCBA定制流程

Created2022-05-19|Updated2022-07-01|定制商务合作
|Post View:

Y&R团队PCBA定制服务

团队目前开放小批量定制PCB/PCBA，我们有丰富的客制化键盘PCB制造经验，较小的起订量（MOQ大于或等于50），能为您提供高质量的定制服务。您可联系我们团队任意成员进行咨询和预报价，在进行咨询前我们非常建议您仔细阅读下面的相关条款和注意事项。

可定制产品规格

功能规格

请确认您需要定制的键盘产品功能符合以下几个要求之一，您也可以提供样板我们会为您评估可行性：

  1. 单模键盘PCB，使用QMK固件，VIA作为改建软件。附加外设列表为：
    1. RGB轴灯和底灯，一般采用2812或IS31FL37xx驱动，轴灯使用3528封装的反贴灯
    2. 旋转编码器即旋钮，一般为EC11规格
    3. OLED黑白显示屏，一般是4线控制，常规尺寸为0.91寸和1.3寸
    4. 摇杆，一般是4线控制（带按压的为5线）
    5. 彩色屏幕，一般支持驱动型号为st77xx，ILI9xxx，GC9A01，SSD1351
  2. 双模键盘PCB，支持蓝牙和USB连接，使用Y&R修改版的QMK作为固件，VIA或Y&R研发的KBCTL-NG作为改建软件。可附加外设同单模PCB。蓝牙关键参数为：
    1. 蓝牙芯片为NRF528xx，一般使用NRF52810，支持蓝牙5.x
    2. 蓝牙可支持最多8个无线设备，一般开放4个供玩家使用
    3. 蓝牙下具有全键无冲功能
    4. 蓝牙下具有1k以上的扫描频率，蓝牙默认回报率为125HZ，实测蓝牙触发最小时间间隔为1-2ms，满足日常使用和大部分游戏使用要求
    5. 蓝牙下按键和USB下按键键位一致，可通过驱动进行修改，使用VIA改建时需要在USB连线时才能修改

外观和硬件规格

请确认您需要定制的键盘产品外观符合以下每项要求之一，您也可以提供样板我们会为您评估可行性：

  1. 轴体安装方式
    1. 热拔插，热拔插轴座提供kaihua，jwk，jiadalong可选，颜色kaihua，jiadalong一般为黑色，jwk一般为白色并可选彩色
    2. 焊接
  2. 配列
    1. 焊接一般可以支持大部分配列，并且可以一板兼容多个配列
    2. 热拔插一般建议只做一个配列，兼容多配列存在以下问题，请慎重考虑
      1. 部分轴座需要旋转90度排列，插上轴后可能会导致部分键帽安装过紧甚至裂开底部十字柱的可能
      2. 部分轴座会遮挡5脚轴的轴孔，会有1-2个轴需要减脚后才能顺利安装
      3. 部分轴座位于PCB卫星轴的中间，如果需要使用卫星轴垫片（一般在1.2的板厚PCB中常用）会存在干涉问题
  3. 板材以及阻焊
    1. 板材提供FR4纤维板材，一般有下面2种规格：
      1. 普通半透明浅黄色玻璃纤维
      2. 全黑色不透明黑芯板材。这类板材较贵，且订货周期较长
    2. 焊盘可选沉金和喷锡2种
    3. 阻焊颜色分为3种，一般可选为：
      1. 普通色：哑光黑，白，红，蓝，绿，紫，黄。这类色较为便宜
      2. 透明色：全透明亮油，半透明灰油。这类色较贵，一般搭配黑芯板材使用
      3. 特殊色：粉色，灰色，哑光绿等。这类色是客户指定色或者厂家特有色，价格较贵，工期较长
    4. PCB厚度可选：1.6mm, 1.2mm, 0.8mm
    5. 目前团队支持市面上常见的大部分开槽板材设计，包括但不限于多个横槽，多个竖槽，单按键开槽以及组合槽等
  4. 插件和双面贴片
    1. 一般常规物料都是贴片封装并且在PCB的背面，PCB正面不放元器件，部分插件有时需要放正面，如果量少会采用人工焊接的方式，贴片费用会相对较高，这部分可选出厂焊接完成或者您自行处理，需要插件或者双面贴片的物料常见的有：
      1. 旋钮，插件，且需要放正面
      2. OLED，插件，且需要放正面
      3. TYPEC母座，某些情况需要放在PCB正面，需要双面贴片（会多开钢网和胶网）
      4. 蓝牙接口，某些情况需要放在PCB正面，需要双面贴片
  5. 发货清单
    1. 单模键盘一般包括：pcb主板，放置pcb主板的防静电袋，pcb小板，pcb线材，放置pcb小板的防静电袋
    2. 双模键盘除上述单模键盘配件之外，一般还包括：蓝牙模块，蓝牙模块使用的线材，放置蓝牙模块的防静电袋
    3. 批量发货时pcb为单独使用防静电袋包装，一般3-5片pcb打包为一盒，使用硬纸盒包装发货

定制要求

目前团队人员和资源较为有限，请确保您的PCB定制需求满足以下全部条件：

  1. 单款样式PCB数量最少为50张（一款PCB具有热拔插和焊接2个款式需要保证热拔插PCB数量大于等于50，焊接PCB数量也大于等于50）
  2. 打样时间从交付打样定金开始计算，约定交期不低于15天
  3. 打样完成后20天内下单并结算小批量订单
  4. 小批量订单从下单并支付生产款开始计算，约定交期不低于20天
  5. 由于生产事故造成的交货数量不足，并且不足数量低于定制数量的5%，按照生产报价的单价*不足数量进行赔偿。不足数量超过5%时，将会在30天内进行二次生产补足预定的交货数量
  6. 生产数量需要留有冗余量以供售后，建议冗余量为5%-10%的定制数量
  7. 出货后所有PCB相关的售后服务和开销均由Y&R团队负责和承担

收费标准

目前我们分为2个阶段收费，分别是打样阶段和小批量生产阶段，各阶段收费标准为：

打样阶段

在确认功能和外形以及接受团队给出的预报价后，需要支付2000元人民币作为预付定金，在进行小批量下单后的24小时内，我们会结算打样的开销并退还您预付定金中剩余的款项。
2000的预付款中包括以下几部分开销：

  1. 基础设计费350元
  2. 附加功能设计费，以下功能和外形，每添加一项收取对应的费用一次：
    1. RGB功能，50元
    2. PCB开单一横槽或者单一竖槽，50元
    3. PCB单按键开槽或者复杂的组合槽，100元
    4. 双模PCB，50元
  3. 打样板材费用，按照实际支出收取，最多打样3次。
  4. 打样板子的物料费用，按照实际支出收取，一般单模70，双模100
  5. 打样的收工调试和焊接费，带RGB的收取80元，不带RGB的收取60元

在完成打样后并下单小批量订单后，我们会按照您小批量下单的数量给予设计费优惠，板子数量从100片开始计算，每超过50片一次，设计费少收50一次，最多优惠350元。

注意事项：只有在小批量下单后才会退还剩余打样预付款，并且小批量订单要满足定制要求。

小批量阶段

小批量阶段我们采用成本价+抽成价的方式进行收费，其中成本价为所有物料和加工费的总和（我们会在报价单中清晰地列出每一项的花费），抽成部分按照小批量的数量实行阶梯价收费，具体收费标准如下：

数量 每片抽成费用
[50,100) 50元
[100,300) 40元
[300,500) 35元
[500,+inf） 30元

所有Y&R承接的定制订单，均附加个人售后服务，您可将一切PCB问题交付我们团队处理，出货后也可安排Y&R团队成员进客户群处理售后问题，具体售后标准详见，Y&R售后说明。
